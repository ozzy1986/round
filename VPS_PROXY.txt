VPS PROXY — QUICK REFERENCE
===========================

What: Squid HTTP proxy on the VPS. Only one IP is allowed (yours). Use it in browser or app to reach blocked sites (e.g. Google).

USE THE PROXY
-------------
Proxy server:  162.246.19.78
Port:          3128
Type:          HTTP

- Firefox: Settings → Network Settings → Manual proxy → HTTP Proxy 162.246.19.78, Port 3128.
- Chrome / Windows: Settings → System → Open proxy settings → Use proxy server On → Address 162.246.19.78, Port 3128.
- Gradle (when Google is blocked): already set in android/gradle.properties (systemProp.http.proxyHost etc.).


WHEN YOUR IP CHANGES — UPDATE ALLOWED IP
---------------------------------------
1. SSH to VPS:
   ssh root@162.246.19.78

2. Update firewall (UFW) — replace NEW_IP with your current IP (e.g. 192.166.101.44):
   ufw delete allow from OLD_IP to any port 3128 proto tcp
   ufw allow from NEW_IP to any port 3128 proto tcp

   Or edit and reload:
   sed -i 's/OLD_IP/NEW_IP/g' /etc/ufw/user.rules
   ufw reload

3. Update Squid ACL — replace NEW_IP in:
   sed -i 's#acl myhome src OLD_IP/32#acl myhome src NEW_IP/32#' /etc/squid/squid.conf
   squid -k reconfigure

4. Check:
   ufw status | grep 3128
   grep myhome /etc/squid/squid.conf

Done. Only the new IP can use the proxy.
